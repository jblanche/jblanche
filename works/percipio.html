Percipio

Between October 2016 and July 2018 I worked for Skillsoft to help them create the Percipio learning platform.

Quickly, I got addicted to React.js and the component approach I was already trying to pursue in previous missions. I also liked the Backend for Frontend approach that was made in Node.js to aggregate and filter calls to lower APIs. All along, I was happilly working with both the UI and back-end teams and felt part of the teams in the decisions making.

I worked both on the learner and admin parts of the app. From the Node.js API to the UI Storybook.
My missions went from creating the firsts courses tests for learners, to replacing the Auth0 login UI with our own and to add a customization option to the interface so that clients could have an interface matching their brand design…
When I thought our JavaScript bundles became too big, I also took the leadership in the Webpack build optimisation, resulting in big performance improvements and customers satisfaction.
